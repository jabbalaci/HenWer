HenWer 0.1h

HenWer is a viewer for online and local images. HenWer was written in Python using the wxPython graphical library.

HenWer is a fork of ManWer. ManWer was originally written by Matteo Bana, however it seems that he abandoned his project.

HenWer adds some new (and hopefully useful) features. HenWer is developed by Jabba Laci (jabba.laci@gmail.com). The source code is available at GitHub.
