Key Bindings

M or F11 maximize window (full screen)
+ or ] zoom in
- or [ zoom out
F fit image to window size
= set zoom to 100%
H help (you see that)
I image information
G goto image
R random jump
Y yank (copy) image's absolute path to clipboard
S mark image to be saved
D mark local file to be deleted
W mark image to be saved as wallpaper
A mark all images to be saved
C commit changes (save, delete, etc.)
